# **App Name**: ASC Khombole

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth, supporting email/password and Google sign-in.
- Content Management: Back-office admin panel for CRUD (Create, Read, Update, Delete) operations on all content.
- News Feed: Display a blog-style news feed with images, tags, and a comment system.
- Match Calendar: Display a calendar of upcoming and past matches, including scores, location, and stats.
- Team Roster: Show a grid of player profiles with photos, stats, and detailed individual pages.
- Media Gallery: Showcase photo and video albums, with upload capabilities for admins.
- Sponsor Showcase: Feature a dedicated section for sponsors, including logos, descriptions, and a 'Become a Partner' form.

## Style Guidelines:

- Primary color: The club's iconic green (#0EA75A) evokes the team's core identity.
- Background color: A very light, desaturated green (#F0F8F4) provides a clean, sporty backdrop, allowing content to stand out while maintaining brand consistency.
- Accent color: Gold (#CFA10A), used sparingly, provides highlights for important elements and calls to action, conveying prestige.
- Body and headline font: 'Inter' (sans-serif) will create a modern, sporty feel.
- Use soccer-themed icons for key navigation elements, creating a clear visual language for users to easily identify each section.
- Employ a modern, dynamic layout inspired by Toulouse FC's website, featuring full-screen images, carousels, and well-structured sections to keep users engaged and informed.
- Incorporate subtle animations and transitions on content load and user interactions to enhance the dynamic feel.
